Mon projet Web : Hôtel Chambord

Présentation du projet

Le projet consiste en un site web pour l'Hôtel Chambord, un établissement fictif, avec une interface moderne et responsive. L'objectif est de présenter les chambres, les services, et de permettre aux visiteurs de réserver en ligne.

  • Site web responsive adapté aux mobiles et tablettes
  • Pages : Accueil, Chambres, Services, Réservation, Contact
  • Formulaire de réservation fonctionnel avec validation des données
  • Galerie d’images pour mettre en valeur les chambres et les espaces communs
  • Navigation claire et expérience utilisateur optimisée

Ce projet m’a permis de renforcer mes compétences en HTML, CSS, JavaScript et en design web interactif.
